数组： 
prepend: O(1), append: O(1), insert：O(n), del: O(n), lookup:O(1)
链表：
prepend: O(1), append: O(1), insert：O(1), del: O(1), lookup:O(n)
跳表：
对标平衡树，和二分查找，一定要是有序序列，通过升维，添加多级索引，空间换时间来减少链表的查询时间，O(logn)
空间复杂度位O(n)

insert：O(1), del: O(1)，lookup: O(n)
队列(queue)：先入先出
insert：O(1), del: O(1), lookup: O(n)
